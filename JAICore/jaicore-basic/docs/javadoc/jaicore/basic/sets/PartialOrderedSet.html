      • getTotalOrder

        public java.util.List<E> getTotalOrder()
        Creates a total order of the elements stored in this PartialOrderedSet. The order is created by iterating over all elements in the set and inserting each element in the list at the position of the element with the currently smallest index that has to be after the current element. If no elements needs to be after the current element, it is appended to the end. The runtime is O(nÂ²).
        Returns:
        A total ordering of the elements in this PartialOrderedSet.
